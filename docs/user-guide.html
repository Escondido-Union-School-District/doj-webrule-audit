DOJ WebRule Audit

User Guide — Escondido Union School District

What This Tool Does

Audits all EUSD district webpages for ADA/WCAG compliance as required by the DOJ web accessibility rule. Tracks remediation progress across the district so you can see which pages have been fixed and which still need attention.
